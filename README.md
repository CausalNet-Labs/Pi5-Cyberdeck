# Pi 5 Cyberdeck Engineering Manual

Engineering handbook for a Raspberry Pi 5 cyberdeck using an Apache 2800 enclosure.

Locked hardware: Raspberry Pi 5 (16 GB) · Official Active Cooler · Official M.2 HAT+ · 1 TB NVMe SSD ·
Waveshare 8″ IPS HDMI Capacitive Touchscreen (1280×800) · Razer Huntsman Mini (60%) ·
Official 27W USB-C Power Supply · Apache 2800 Protective Case.

## Start here

**[docs/README.md](docs/README.md)** — full chapter index (15 chapters, chapter-opening blueprint plates included).

## Structure

- `docs/` — the 15-chapter manual, converted from the source PDF (`docs/chapters/`), plus the index.
- `assets/diagrams/` — the 15 blueprint plates (cover + one per chapter, `plate-000.png`–`plate-014.png`), embedded inline in the chapters.
- `templates/` — standalone printable worksheets (measurement, cable, build log) pulled out of the Chapter 15 appendices.
- `scripts/` — helper shell scripts distilled from the manual's command references (`check-thermals.sh` from Ch.3/8, `setup-dev-environment.sh` from Ch.10).
- `.github/` — CI workflow that link-checks the docs on push/PR.

## Notes on the conversion

Chapters were generated by extracting and reflowing the text of
`Pi5_Cyberdeck_Engineering_Manual_V2_Integrated.pdf`. Headings, bullet
lists, shell command blocks, and most tables converted cleanly. A few
large multi-column tables — notably in Chapter 14 — kept minor row-grouping
artifacts from the PDF layout; the underlying text is complete, just
double-check those specific tables against the source PDF if something
looks misaligned.

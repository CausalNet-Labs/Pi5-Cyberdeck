# Build Journal Template

Use this structure for every entry in your build journal (paper or
digital), starting from Chapter 1.7. Copy the block below per entry.

```
Date:
Chapter / section reference:
What I measured (number, tool used, part revision):
What I changed from the plan, and why:
Photo reference (filename or description):
Open questions / things to verify before the next irreversible step:
```

## Photo checklist (Chapter 15.7)

- [ ] Bench assembly, before case work (Chapter 3)
- [ ] Paper templates dry-fit in the case (Chapter 4.5)
- [ ] Finished cut opening, before mounting (Chapter 5.10)
- [ ] Fully mounted interior, before final cable dress (Chapter 6.7)
- [ ] Final assembled interior with cabling secured (Chapter 7.10)
- [ ] Closed, latched exterior — the finished device
